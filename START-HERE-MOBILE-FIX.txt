WHAT WAS WRONG:
===============

Desktop CSS used:
  - Fixed viewport heights (height: 100vh)
  - Hidden overflow (overflow-y: hidden)

Result on mobile:
  - Content stacked vertically
  - Container had fixed height
  - Overflow was cut off (not scrollable)
  - You saw only what fit in viewport

THE FIX:

  - Containers expand with content
  - All content accessible via scroll
